Sobre Nós

Dary é uma loja de moda Online criada em 2024 por Dary Gustavo. Ela começou puramente como uma plataforma para integrar tecnologias de desenvolvimento back-end e exibi-las no meu GitHub para empregadores em potencial e aprimorar minhas habilidades no uso de tecnologias.

À medida que este site se desenvolve, ele passará por mais mudanças, como a adição do framework Bootstrap para responsividade, inclusão de um banco de dados de produtos e um carrinho totalmente funcional e portal de conta de usuário.
